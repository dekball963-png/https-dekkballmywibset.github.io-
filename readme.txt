alexcandra-website — petunjuk singkat

Isi folder:
- index.html
- style.css

Cara pakai:
1. Unggah file index.html dan style.css ke repository GitHub (repo: dekkbalmywibset).
2. Commit perubahan.
3. Aktifkan GitHub Pages (Settings → Pages → Deploy from a branch → main / root).
4. Tunggu ~1-2 menit, lalu buka: https://dekkbalmywibset.github.io/

Catatan:
- Untuk mengganti teks, edit index.html (judul, lirik, kontak).
- Pastikan style.css ada di folder yang sama.
